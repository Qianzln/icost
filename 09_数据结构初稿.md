# 09. 数据结构初稿

## 1. User 用户

```json
{
  "id": "user_001",
  "nickname": "小明",
  "avatarUrl": "https://example.com/avatar.png",
  "phone": "13800000000",
  "loginProvider": "phone",
  "createdAt": "2026-06-01T10:00:00Z",
  "updatedAt": "2026-06-01T10:00:00Z"
}
```

## 2. Ledger 账本

```json
{
  "id": "ledger_001",
  "name": "我们的小家",
  "type": "couple",
  "currency": "CNY",
  "coverIcon": "home",
  "ownerId": "user_001",
  "defaultAccountId": "account_001",
  "defaultSplitMethod": "equal",
  "createdAt": "2026-06-01T10:00:00Z",
  "updatedAt": "2026-06-01T10:00:00Z"
}
```

账本类型：

- personal：个人账本
- couple：情侣账本
- family：家庭账本
- roommate：合租账本
- travel：旅行账本
- project：专项账本

## 3. LedgerMember 账本成员

```json
{
```

角色：

- owner：创建者
- admin：管理员

- viewer：访客

```json
  "name": "微信钱包",
  "type": "asset",
```

- asset：资产

账户归属：

- shared：共同

## 6. Transaction 流水

```json
{
  "id": "txn_001",
  "ledgerId": "ledger_001",
  "type": "expense",
  "amount": 100.00,
  "currency": "CNY",
  "categoryId": "cat_001",
  "accountId": "account_001",
  "creatorUserId": "user_001",
  "payerUserId": "user_001",
  "scope": "shared",
  "occurredAt": "2026-06-01T12:30:00Z",
  "note": "午餐",
  "tags": ["约会"],
  "imageUrls": [],
  "isPrivate": false,
  "privacyLevel": "public",
  "isIncludedInBudget": true,
  "createdAt": "2026-06-01T12:31:00Z",
  "updatedAt": "2026-06-01T12:31:00Z"
}
```

流水类型：

- expense：支出
- income：收入
- transfer：转账
- reimbursement：报销
- borrow_in：借入
- borrow_out：借出

支出范围：

- personal：个人支出
- shared：共同支出

隐私级别：

- public：公开
- partial：半私密
- private：私密

## 7. TransactionParticipant 消费对象/参与人

```json
  "role": "consumer",
  "createdAt": "2026-06-01T12:31:00Z"

```

角色：

- payer：付款人
- consumer：消费对象
- bearer：承担人

## 8. Split 分摊

```json
{
  "id": "split_001",
  "transactionId": "txn_001",
  "ledgerId": "ledger_001",
  "method": "equal",
  "createdAt": "2026-06-01T12:31:00Z",
  "items": [
    {
      "userId": "user_001",
      "shouldPay": 50.00,
      "actualPaid": 100.00
    },
    {
      "userId": "user_002",
      "shouldPay": 50.00,
      "actualPaid": 0.00
    }
  ]
}
```

分摊方式：

- equal：平均分
- ratio：按比例
- custom：指定金额
- single：一人承担
- none：不分摊

## 14. 数据更新规则

新增、编辑、删除流水时，需要同步更新：

- 账户余额
- 月度收支统计
- 分类统计
- 成员统计
- 预算使用情况
- 待结算金额
- 共同目标进度，可选
