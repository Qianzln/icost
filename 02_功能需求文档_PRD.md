# 02. 功能需求文档 PRD

## 1. 产品概述

小家账本是一款移动端共享记账 App，核心功能包括：共同账本、快速记账、成员管理、分摊结算、账单流水、资产管理、统计分析、预算管理和设置中心。

## 2. 角色与权限

| 角色 | 权限 |
|---|---|
| 创建者 | 创建账本、删除账本、管理成员、管理权限、导出数据 |
| 管理员 | 管理分类、账户、预算、部分成员设置 |
| 成员 | 新增/编辑本人创建的账单，查看授权范围内的共同账单 |
| 访客 | 只读查看被授权的账本数据 |

## 3. 核心功能模块

### 3.1 用户系统

优先级：P0

功能：

- 手机号登录
- 第三方登录：微信、Apple ID，可按平台选择
- 昵称、头像设置
- 本地密码保护
- 生物识别解锁
- 多设备同步

基础字段：

- 用户 ID
- 昵称
- 头像
- 手机号
- 登录方式
- 创建时间

### 3.2 账本管理

优先级：P0

账本类型：

- 个人账本
- 情侣账本
- 家庭账本
- 合租账本
- 旅行账本
- 装修账本
- 宝宝账本

功能：

- 创建账本
- 切换账本
- 编辑账本名称、图标、币种
- 归档账本
- 删除账本
- 设置默认账本
- 设置默认记账账户
- 设置默认分摊规则

### 3.3 成员管理

优先级：P0

功能：

- 邀请成员加入账本
- 通过链接、二维码或手机号邀请
- 设置成员昵称
- 设置成员头像
- 设置成员角色
- 移除成员
- 成员退出账本
- 查看成员贡献：记账次数、付款金额、承担金额

成员字段：

- 创建人
- 付款人
- 消费对象
- 承担人

### 3.4 快速记账

优先级：P0

支持账单类型：

- 支出
- 收入
- 转账
- 报销
- 借入
- 借出

支出字段：

| 字段 | 是否必填 | 说明 |
|---|---|---|
| 金额 | 是 | 支出金额 |
| 分类 | 是 | 如餐饮、购物、住房 |
| 账户 | 否 | 默认账户可自动带出 |
| 时间 | 否 | 默认为当前时间 |
| 备注 | 否 | 用户手动填写 |
| 标签 | 否 | 如旅行、约会、宝宝 |
| 图片 | 否 | 账单截图或发票 |
| 付款人 | 否 | 默认当前用户 |
| 消费对象 | 否 | 我、对方、两人、家庭、宝宝等 |
| 支出归属 | 否 | 个人支出或共同支出 |
| 分摊方式 | 否 | 平均分、按比例、指定金额、一人承担 |

### 3.5 流水管理

优先级：P0

功能：

- 月度流水列表
- 按日期分组
- 搜索账单
- 按分类筛选
- 按成员筛选
- 按账户筛选
- 按标签筛选
- 流水详情
- 编辑流水
- 删除流水
- 复制为模板

列表展示字段：

- 分类图标
- 分类名称
- 备注
- 付款人头像
- 是否共同支出
- 账户名称
- 金额

### 3.6 分摊结算

优先级：P0

功能：

- 平均分摊
- 一人承担
- 按比例分摊
- 指定金额分摊
- 自动生成待结算金额
- 查看“我应还对方”
- 查看“对方应还我”
- 一键标记已结算
- 查看结算历史

MVP 建议先支持：

- 双人平均分
- 一人承担
- 待结算展示
- 标记已结算

## 4. 非功能需求

### 4.1 性能

- 首页首屏加载目标：2 秒以内
- 快速记账面板打开目标：300 毫秒以内
- 流水列表滑动保持流畅

### 4.2 安全

- HTTPS 传输
- 敏感数据加密存储
- 私密账单权限隔离
- 支持本地应用锁
- 数据导出需二次确认

### 4.3 易用性

- 新用户 1 分钟内完成第一笔记账
- 常用分类前置
- 常用字段自动带出默认值
- 高级字段折叠，避免干扰快速记账

## 5. 核心指标

| 指标 | 目标 |
|---|---|
| 首日记账率 | 大于 50% |
| 邀请成员转化率 | 大于 30% |
| 7 日留存率 | 大于 25% |
| 周记账次数 | 人均大于 5 次 |
| 分摊功能使用率 | 大于 20% |
| 预算设置率 | 大于 15% |
