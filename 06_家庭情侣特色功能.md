# 06. 家庭情侣特色功能

## 1. 共同账本

共同账本是产品核心。用户可以创建一个只属于两个人或家庭成员的账本，用来记录共同生活中的收入、支出、转账、分摊和资产。

### 核心能力

- 多成员共同记账
- 每笔账记录创建人、付款人、消费对象、承担人
- 共同支出和个人支出分开
- 账单支持私密
- 成员维度统计
- 自动生成待结算金额

### 推荐账本模板

| 模板 | 默认分类 |
|---|---|
| 情侣账本 | 餐饮、约会、旅行、礼物、交通、购物 |
| 家庭账本 | 买菜、住房、水电、日用、宝宝、医疗、车贷 |
| 合租账本 | 房租、水电、燃气、宽带、公共用品 |
| 旅行账本 | 交通、住宿、餐饮、门票、购物、保险 |
| 装修账本 | 建材、家具、家电、人工、设计、软装 |

## 2. 成员维度记账

每笔账单建议包含四个成员维度：

| 字段 | 说明 |
|---|---|
| 创建人 | 谁记录了这笔账 |
| 付款人 | 谁实际支付了钱 |
| 消费对象 | 这笔钱花在谁身上 |
| 承担人 | 最终由谁承担这笔支出 |

示例：

```text
账单：晚餐 ¥200
创建人：我
付款人：我
消费对象：两人
承担人：我 ¥100，对方 ¥100
```

## 3. 自动分摊结算

### 支持方式

- 平均分
- 按比例
- 指定金额
- 一人承担
- 不分摊，仅记录

### 结算结果

系统自动汇总为：

- 我应还对方
- 对方应还我
- A 应还 B
- B 应还 C
- 多成员净额抵消后结果

### 多笔账单合并结算

系统不需要每笔账单单独还款，而是按月或按当前未结算金额合并展示。

示例：

```text
本月未结算
对方需还你 ¥940

由 12 笔共同账单产生
```

## 4. 私密账单

情侣和家庭记账必须支持隐私。

### 私密等级

| 等级 | 说明 |
|---|---|
| 公开 | 成员可查看完整账单 |
| 半私密 | 成员只看到金额和分类，不看到备注、图片 |
| 私密 | 只有本人可见 |
| 不计入共同账本 | 只进入个人账本，不进入共同统计 |

### 适用场景

- 礼物
- 医疗
- 个人购物
- 临时借款
- 不想公开的消费

## 5. 共同预算

家庭或情侣可共同设置预算。

### 预算类型

- 月度总预算
- 分类预算
- 成员预算
- 共同支出预算
- 专项预算

### 预算提醒

- 预算使用达到 80%：轻提醒
- 预算使用达到 100%：超支提醒
- 周报：本周花了多少、还剩多少

### 示例

```text
餐饮预算 ¥3000
已使用 ¥2580
剩余 ¥420
使用率 86%
提示：本月餐饮支出偏高
```

## 6. 共同目标

适合长期储蓄和家庭计划。

### 目标类型

- 旅行基金
- 买房基金
- 婚礼基金
- 装修基金
- 宝宝基金
- 宠物基金
- 应急备用金

### 字段

- 目标名称
- 目标金额
- 当前金额
- 截止日期
- 参与成员
- 每月建议存入金额
- 进度

### 示例

```text
旅行基金
目标：¥20,000
已存：¥6,800
进度：34%
预计还需 5 个月完成
```

## 7. 关系账单

这是产品的情感化差异点，可以放在统计页。

### 可展示内容

- 本月共同消费总额
- 本月约会花费
- 本月谁付款更多
- 本月谁记账更多
- 本月待结算金额
- 本月共同目标存入金额
- 本月固定支出占比

### 文案示例

```text
这个月你们一起花了 ¥7,100。
你为共同账单多付了 ¥940。
餐饮支出占共同支出的 32%。
你们的旅行基金已完成 34%。
```

## 8. 家庭财务周报 / 月报

系统可以自动生成家庭财务报告。

内容包括：

- 本周/本月总支出
- 分类 TOP 5
- 成员付款情况
- 待结算金额
- 预算剩余
- 异常支出提醒
- 共同目标进度

## 9. 成员权限

| 权限项 | 创建者 | 管理员 | 成员 | 访客 |
|---|---|---|---|---|
| 查看共同账单 | 是 | 是 | 是 | 是 |
| 新增账单 | 是 | 是 | 是 | 否 |
| 编辑他人账单 | 是 | 可配置 | 否 | 否 |
| 删除账本 | 是 | 否 | 否 | 否 |
| 导出数据 | 是 | 可配置 | 否 | 否 |
| 查看私密账单 | 仅本人 | 仅本人 | 仅本人 | 否 |
