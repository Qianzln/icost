# 05. 记账流程与交互说明

## 1. 新增支出流程

```text
点击 +
→ 选择支出
→ 选择分类
→ 输入金额
→ 选择账户
→ 选择付款人
→ 选择消费对象
→ 选择共同支出/个人支出
→ 设置分摊方式
→ 填写备注、标签、图片
→ 点击完成
→ 返回账本首页
→ 刷新流水、统计、预算、待结算
```

## 2. 最简记账路径

为了保证高频使用体验，用户可以只填写：

```text
分类 + 金额
```

系统自动带出：

| 字段 | 默认值 |
|---|---|
| 时间 | 当前时间 |
| 账户 | 默认账户 |
| 付款人 | 当前用户 |
| 支出归属 | 上次选择或账本默认设置 |
| 分摊方式 | 账本默认分摊规则 |

## 3. 新增收入流程

```text
点击 +
→ 选择收入
→ 选择收入分类
→ 输入金额
→ 选择入账账户
→ 选择收入归属：我的/对方/共同
→ 填写备注
→ 完成
```

收入分类建议：

- 工资
- 奖金
- 兼职
- 理财
- 红包
- 报销收入
- 其他收入

## 4. 新增转账流程

```text
点击 +
→ 选择转账
→ 选择转出账户
→ 选择转入账户
→ 输入金额
→ 选择手续费，可选
→ 填写备注
→ 完成
```

转账不计入收入和支出，但影响账户余额。

## 5. 分摊流程

### 5.1 平均分摊

适用场景：情侣吃饭、合租水电、家庭共同支出。

```text
输入账单金额
→ 选择共同支出
→ 选择参与成员
→ 选择平均分
→ 系统自动计算每人承担金额
→ 保存账单
→ 生成待结算记录
```

示例：

```text
账单金额：¥100
付款人：我
参与人：我、对方
分摊方式：平均分

我应承担：¥50
对方应承担：¥50
我实际支付：¥100
对方需还我：¥50
```

### 5.2 一人承担

适用场景：个人消费、礼物、某人专属支出。

```text
输入金额
→ 选择消费对象
→ 选择一人承担
→ 保存账单
→ 不生成待结算
```

### 5.3 按比例分摊

适用场景：夫妻按收入比例分摊房租、合租按房间面积分摊。

```text
输入金额
→ 选择参与成员
→ 输入比例
→ 系统计算承担金额
→ 保存账单
```

示例：

```text
房租 ¥5000
我承担 60%：¥3000
对方承担 40%：¥2000
```

### 5.4 指定金额分摊

适用场景：其中一人承担固定金额，其余由另一方承担。

规则：

- 每个成员手动输入承担金额
- 系统校验合计金额必须等于账单金额
- 如不等，禁止保存并提示修改

## 6. 待结算生成规则

系统根据以下字段计算：

- 付款人实际支付金额
- 每个成员应承担金额
- 每个成员已支付金额

计算公式：

```text
待结算金额 = 应承担金额 - 已支付金额
```

当某成员待结算金额大于 0，说明该成员需要向付款多的一方结算。

## 7. 编辑账单流程

```text
点击流水
→ 进入流水详情
→ 点击编辑
→ 修改金额、分类、账户、付款人、分摊方式等
→ 点击保存
→ 系统重新计算账户余额、统计、预算、待结算
```

编辑规则：

- 修改金额会重新计算分摊
- 修改付款人会重新计算待结算
- 修改分类会重新计算分类统计和预算
- 修改账户会更新账户余额

## 8. 删除账单流程

```text
点击流水
→ 进入流水详情
→ 点击删除
→ 二次确认
→ 删除账单
→ 回滚账户余额、统计、预算、待结算
```

删除提示文案：

```text
删除后将同时影响统计、预算和待结算金额，确认删除吗？
```

## 9. 周期账单流程

适用场景：房租、水电、会员订阅、保险、房贷、车贷。

```text
创建周期账单
→ 设置分类、金额、账户、周期、开始日期
→ 设置是否自动入账
→ 设置提醒时间
→ 到期后提醒或自动生成账单
```

周期类型：

- 每天
- 每周
- 每月
- 每年
- 自定义周期

## 10. 交互原则

1. 高频字段放前面，低频字段折叠。
2. 记账完成后给轻提示，不打断用户。
3. 分类支持记忆排序，常用分类自动前置。
4. 分摊结果要实时预览。
5. 金额输入支持加减乘除，方便多人账单计算。
6. 重要操作需要二次确认。
7. 支持“保存再记”，适合一次性录入多笔账。
