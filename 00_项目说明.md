# 00. 项目说明

## 项目名称

暂定名称：**小家账本**

这是一款专门面向**家庭、情侣、夫妻、合租伙伴**的共享记账软件。产品参考用户提供截图中的成熟记账软件形态：

- 卡片式首页
- 底部悬浮导航
- 快速记账弹窗
- 资产总览
- 统计看板
- 设置中心
- 分类、标签、预算、周期账单等完整配置能力

在此基础上，本产品重点强化“共同生活财务管理”场景，包括共同账本、成员记账、分摊结算、共同预算、家庭资产、私密账单和共同目标。

## 产品核心目标

让家庭或情侣之间的日常消费、共同支出、个人支出、分摊结算、资产负债和预算目标变得清楚、透明、低成本。

## 产品核心原则

1. **记账要快**：高频记账路径控制在 3 步以内。
2. **共同账要清楚**：谁花的、谁付的、谁承担的要清晰。
3. **结算要自动**：共同支出自动生成待结算金额。
4. **隐私要可控**：个人账单、私密账单和共同账单分开。
5. **统计要有结论**：不仅展示数字，还要告诉用户预算是否健康、谁多付了、哪里超支了。
