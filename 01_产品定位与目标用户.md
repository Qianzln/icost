# 01. 产品定位与目标用户

## 1. 产品定位

**小家账本**是一款面向家庭、情侣、夫妻和合租伙伴的共享记账软件。

它不仅解决“我花了多少钱”，还解决：

- 我们这个月一起花了多少钱
- 谁为共同生活付得更多
- 哪些支出需要分摊
- 哪些支出属于个人消费
- 月底谁应该还谁多少钱
- 家庭预算是否超支
- 家庭资产、负债、净资产现在是多少

一句话定位：

> 帮情侣和家庭把共同花销、分摊结算、预算资产都记清楚。

## 2. 目标用户

### 2.1 情侣

典型场景：

- 日常吃饭、看电影、购物、打车
- 节日礼物、旅行、约会消费
- 双方轮流付款但月底想知道是否公平
- 部分消费 AA，部分消费一方承担

核心需求：

- 快速记录共同支出
- 自动计算谁该还谁
- 记录恋爱期间共同消费
- 不让钱的问题影响关系

### 2.2 夫妻 / 小家庭

典型场景：

- 房租、房贷、水电、物业、买菜、日用品
- 宝宝、宠物、医疗、保险、车贷
- 家庭收入、支出、储蓄和负债管理

核心需求：

- 家庭账本共同维护
- 月度预算共同控制
- 家庭资产统一查看
- 固定支出自动提醒

### 2.3 合租伙伴

典型场景：

- 房租、水电、燃气、宽带、公共用品
- 多人共同付款，月底统一结算

核心需求：

- 共同账单透明
- 自动分摊
- 待结算提醒
- 减少口头沟通成本

### 2.4 多成员家庭

典型场景：

- 夫妻、父母、子女共同管理家庭收支
- 子女帮父母记录开销
- 大家庭共同承担某些支出

核心需求：

- 多成员权限
- 成员消费统计
- 家庭共同支出和个人支出分离

## 3. 用户痛点

| 痛点 | 具体表现 |
|---|---|
| 共同支出记不清 | 一起吃饭、买东西、打车后忘记谁付的钱 |
| 分摊麻烦 | AA、比例分摊、部分承担需要手动算 |
| 预算失控 | 月底才发现餐饮、购物、房租等支出过高 |
| 隐私尴尬 | 个人消费不想全部展示给对方 |
| 账本混乱 | 个人账、家庭账、旅行账混在一起 |
| 资产分散 | 微信、支付宝、银行卡、信用卡、负债分散难统计 |
| 固定账单易忘 | 房租、水电、会员订阅、信用卡还款容易漏记 |

## 4. 产品价值

### 对情侣

- 共同消费清楚，避免因为钱产生误会。
- 分摊自动计算，减少手动算账。
- 可以沉淀恋爱期间的消费记忆，比如旅行、约会、礼物。

### 对家庭

- 家庭收支透明。
- 预算更可控。
- 资产负债更清楚。
- 固定账单、周期支出不容易遗漏。

### 对合租伙伴

- 公共账单公开透明。
- 自动分摊，月底一键结算。
- 减少“谁欠谁多少钱”的沟通成本。

## 5. 产品差异化

相比普通个人记账软件，本产品重点突出：

1. **成员维度**：每笔账都能记录付款人、消费对象和承担人。
2. **共同账本**：天然支持情侣、夫妻、家庭、合租多人共同记账。
3. **分摊结算**：共同支出自动生成待结算金额。
4. **私密账单**：个人消费可独立保存，不影响共同账本关系。
5. **家庭资产**：支持个人资产、对方资产、共同资产、家庭负债。
6. **关系统计**：展示谁多付、共同支出占比、预算剩余等关系型财务数据。
