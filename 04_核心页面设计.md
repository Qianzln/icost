# 04. 核心页面设计

## 1. 账本首页

### 页面目标

让用户快速知道本月家庭或情侣账本的收支情况，并能快速搜索、筛选、查看和新增账单。

### 页面结构

```text
顶部区域
├── 账本名称：默认账本 / 我们的小家
├── 切换账本按钮
├── 筛选按钮
└── 成员头像入口

内容区域
├── 搜索框
├── 月份切换
├── 月度总览卡片
├── 关系财务卡片
└── 流水列表

底部区域
└── 底部导航 + 快速记账按钮
```

### 月度总览卡片

字段：

- 总支出
- 总收入
- 月结余
- 预算剩余

示例：

```text
2026年6月
总支出 ¥8,520.00
总收入 ¥18,000.00
月结余 ¥9,480.00
预算剩余 ¥1,480.00
```

### 家庭/情侣特色卡片

字段：

- 共同支出
- 个人支出
- 待结算金额
- 谁付款更多

示例：

```text
本月共同支出 ¥7,100
你为共同账单多付 ¥940
对方需还你 ¥940
```

### 流水列表

列表项字段：

- 分类图标
- 分类名称
- 备注
- 付款人
- 支出归属：共同/个人/私密
- 账户
- 金额

示例：

```text
今天 6月26日

餐饮   午餐
共同支出 · 付款人：我 · 默认账户
- ¥58.00

住房   房租
家庭支出 · 付款人：对方 · 银行卡
- ¥3,000.00
```

### 空状态

文案建议：

```text
本月暂无账单
点 + 记录第一笔家庭支出吧
```

## 2. 快速记账页

### 页面目标

在不牺牲记账速度的前提下，支持家庭/情侣场景所需的付款人、消费对象、分摊和标签。

### 页面形态

建议采用底部弹出大面板，接近全屏，类似用户截图中的记账页面。

### 页面结构

```text
顶部
├── 关闭按钮
├── 类型切换：支出 / 收入 / 转账
└── 当前账本

分类区
├── 一级分类
└── 二级分类

快捷字段区
├── 选择账户
├── 付款人
├── 消费对象
├── 分摊
├── 标签
└── 图片

金额区
├── 金额
├── 时间
└── 备注

键盘区
├── 数字键
├── 运算符
├── 删除
├── 保存再记
└── 完成
```

### 分类设计

一级分类建议：

- 餐饮
- 购物
- 服饰
- 日用
- 数码
- 美妆
- 护肤
- 应用软件
- 住房
- 交通
- 娱乐
- 医疗
- 通讯
- 汽车
- 学习
- 宝宝
- 宠物
- 旅行
- 礼物
- 家庭

### 家庭字段

| 字段 | 可选值 |
|---|---|
| 付款人 | 我 / 对方 / 共同账户 / 其他成员 |
| 消费对象 | 我 / 对方 / 两人 / 家庭 / 宝宝 / 宠物 |
| 支出归属 | 个人支出 / 共同支出 / 私密支出 |
| 分摊方式 | 平均分 / 按比例 / 指定金额 / 一人承担 |

### 快速路径

最简记账路径：

```text
选择分类 → 输入金额 → 完成
```

默认值：

- 时间：当前时间
- 账户：默认账户
- 付款人：当前用户
- 归属：上次选择或账本默认值
- 分摊方式：账本默认规则

## 4. 统计页

### 页面目标

帮助用户知道钱花在哪里、谁花得多、预算是否健康、是否需要结算。

### 页面结构

```text
顶部
├── 标题：统计
└── 筛选按钮

时间切换
├── 周
├── 月
├── 年
├── 全部
└── 范围

统计卡片
├── 收支统计
├── 分类统计
├── 成员统计
├── 预算统计
├── 分摊统计
└── 趋势统计
```

### 收支统计卡片

字段：

- 支出
- 收入
- 结余
- 日均支出
- 退款
- 优惠
- 手续费

### 成员统计卡片

字段：

- 我支付
- 对方支付
- 我承担
- 对方承担
- 共同支出
- 个人支出

### 关系结论

统计页应提供类似结论：

```text
本月你为共同账单多支付 ¥940。
本月餐饮支出已达到预算的 86%。
本月共同支出占总支出的 72%。
```

## 6. 待结算页

### 页面目标

集中展示成员之间的应还金额，并支持一键标记已结算。

### 页面结构

```text
顶部
├── 标题：待结算
└── 历史记录

汇总卡片
├── 我应还对方
└── 对方应还我

明细列表
├── 关联账单
├── 金额
├── 付款人
├── 承担人
└── 结算状态
```

### 示例

```text
对方需还你 ¥940

关联账单
- 晚餐 ¥100，对方应还 ¥50
- 打车 ¥80，对方应还 ¥40
- 房租 ¥1,700，对方应还 ¥850

按钮：标记已结算
```
